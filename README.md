## üì¶ Add a Mock Endpoint

### üîπ Request

```http
curl -X POST http://localhost:8080/simulator-admin/addmock \
  -H 'Content-Type: application/json' \
  -d '{
    "path": "/api/hello",
    "method": "GET",
    "responseBody": "{\"msg\":\"Hello, world!\"}",
    "statusCode": 200
  }'
```
##### Alternatively:
mock.json
```
{
  "path": "/api/hello",
  "method": "GET",
  "responseBody": "{\"msg\":\"Hello, world!\"}",
  "statusCode": 200
}
```
```
curl -X POST http://localhost:8080/simulator-admin/addmock \
  -H 'Content-Type: application/json' \
  -d @mock.json
```
